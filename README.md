# Сайт-калькулятор ЛАЙК ФАСАД · редеплой 10.07.2026

## Запуск
Двойной клик на `index.html` — открывается в любом браузере. Интернет нужен только для отправки формы.

## Яндекс Метрика
В `index.html` установлен счётчик **110836947** с Вебвизором, картой кликов и отслеживанием ссылок.

События для целей: `phone_click`, `email_click`, `form_open`, `form_submit`,
`calculator_use`, `halva_interest`, `vk_click`, `whatsapp_click`,
`telegram_click`, `max_click`.

## Включить приём заявок (2 минуты)
1. Зайди на https://web3forms.com, введи почту **i@kuranov-7.ru** → на почту придёт Access Key.
2. Открой `index.html`, найди строку `WEB3FORMS_KEY: ''` (блок CONFIG внизу файла).
3. Вставь ключ между кавычек, сохрани. Всё.

Пока ключ пустой — форма показывает телефон и почту как запасной вариант, заявки не теряются молча.

## Изменить телефон / почту
Тот же блок CONFIG внизу `index.html`:
- `PHONE_DISPLAY` — как показывается
- `PHONE_TEL` — куда звонит клик (формат +7…)
- `EMAIL`

Сейчас стоит **8 (962) 077-84-14** (сверено с фирменным баннером).

## Фото объектов (галерея «до/после» и главное фото)
- Главное фото: положи файл `photos/hero.jpg` — появится в шапке автоматически.
- Карусель до/после: файлы в `photos/`, затем в `index.html` в массив `PHOTOS` добавить строки:
  `{ before: 'photos/obj1-before.jpg', after: 'photos/obj1-after.jpg', label: 'Фасад, Абакан' }`
  Пока массив пустой — секция скрыта.

## Откуда цифры
Все ставки — клиентские «под ключ», откалиброваны движками скиллов (июль 2026):
- Кровля: roofing-estimator, линейный фит по модельным объектам 100–220 м², вилка −10%/+20%
- Фасад сайдинг/панели: facade-estimator, пакеты Эконом→«от», Премиум→«до», точность фита ~1,5%
- Мокрый фасад: prices-2026.md, вилка −5%/+15%
- Забор: формулы zabor-estimator один в один, «от» = Эконом, «до» = Премиум

Внутренних цен (закупка, маржа) в файле НЕТ — только агрегированные клиентские ставки.

## Обновление цен
Когда меняются прайсы в скиллах — попроси Claude пересчитать блок `RATES` в `index.html`
(скрипты калибровки лежат в `dev/`).

## dev/
- `calib_facade.py`, `calib_roof.py` — калибровка ставок движками скиллов
- `zabor_ref.py` — эталонный расчёт забора для сверки с JS (сходится рубль в рубль)
